## Get Started
Start the server by running below command

    >  npm run dev

## Routes used

|Post|Get|Patch|Delete|
|---|---|---|---|

>### Create
**creates single task**

    '/posttask'
>### Read
**fetches all task**
    
    '/alltask'
>### Update
**updates task by ID**
    
    '/updatetask/:id'
>### Delete
**deletes task by ID**
  
    '/deletetaskbyid/:id'
**deletes all task**

    '/deleteAllTask'
